# Blog

Personal blog at https://blog.leoxlin.com

## Requirements

- Install asdf
  
  https://asdf-vm.com/

- Install libs for ruby

  sudo apt-get install libz-dev libssl-dev libffi-dev libyaml-dev

- Install ruby in asdf
  
  asdf install ruby

## Setup

```
gem install jekyll bundler
bundle install
bundle exec jekyll serve
```
